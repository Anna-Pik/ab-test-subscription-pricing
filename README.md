# A/B тест: Оптимизация цены подписки

## Описание проекта

Полный цикл продуктового A/B-теста по изменению цены подписки для новых пользователей.  
Цель эксперимента — повысить конверсию в оплату и увеличить долю подписчиков без ухудшения ключевых метрик монетизации.

## Бизнес-контекст

Около 50% новых пользователей активно использовали продукт, но не совершали платежей.  
При этом бесплатные пользователи получали сопоставимую продуктовую ценность, а текущая стоимость подписки могла создавать ценовой барьер входа.

## Гипотеза

Если снизить цену подписки и сделать её ниже стоимости разовой покупки для новых пользователей, то доля пользователей, выбирающих подписку на раннем этапе, вырастет за счёт более привлекательного ценового позиционирования.

## Что было сделано

### Дизайн эксперимента

- Аудитория: новые пользователи  
- Сплит: постепенный rollout 90/10 → 50/50  
- Контроль: текущая цена подписки  
- Тест: сниженная цена подписки  

### Метрики

**Основная:**
- Доля пользователей с подпиской

**Guardrail metrics:**
- Общая доля платящих пользователей  
- ARPU  
- ARPPU  

**Дополнительные:**
- AOV  
- Конверсия в оплату  
- Структура платежей  

### Аналитическая подготовка

- Расчёт MDE, alpha, beta, sample size  
- Оценка точки безубыточности  
- Проверка рисков каннибализации  
- A/A тесты и симуляции  
- SRM-checks  
- Проверка пересечений с другими экспериментами  

## Результаты

- Доля подписчиков выросла с 20% до 50% (**+30 п.п.**)  
- Конверсия в оплату выросла на **+18.8 п.п.**  
- ARPPU снизился на **27%**  
- AOV снизился на **33%**  
- ARPU статистически значимо не изменился  

## Ключевой вывод

Снижение цены резко увеличило конверсию и долю подписки, но привело к снижению среднего дохода на платящего пользователя и среднего чека.  

Рост произошёл частично за счёт перетока пользователей из разовых покупок в подписку.

## Бизнес-рекомендации

Массовое снижение цены для всей аудитории не рекомендовано.

Оптимальная стратегия:

- сегментные предложения  
- скидка после первой разовой покупки  
- персонализированные офферы  
- ограниченные по времени предложения
